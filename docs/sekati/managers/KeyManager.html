Classpublic class KeyManager
InheritanceKeyManager flash.utils.Proxy

KeyManager provides an interface for the centralized registering and handling of KeyboardEvents via the direct coupling of Keyboard events to actions. Registered Key or Key-Combo's may be added in a repeatable or non-repeatable fashion.

The KeyManger class also proxies the flash.ui.Keyboard and sekati.data.KeyCode classes to provide a single location for KeyCode constants.


Example 
  // create a response relationship between a key and method (non-repeatable):
  KeyManager.$.addKeyListener( [KeyManager.$['UP']], handler );
  
  // create a response between a key combo and a method (repeatable):
  KeyManager.$.addKeyListener( [KeyManager.$['CONTROL'], KeyManager.$['LEFT']], handler, true );

addKeyListener()method
public function addKeyListener(keys:Array, handler:Function, repeat:Boolean = false):void

Add a key or key combo.

Parameters
keys:Array — array of keyCodes (uint's) which will trigger the handler.
 
handler:Function — the function to be fired upon a complete key match.
 
repeat:Boolean (default = false) — if true depressed key handler will fire repeatedly, if false key handler will fire only once per press.

Example 
   KeyManager.$.addKeyListener( [Keyboard.CONTROL, Keyboard.ENTER], runProgram );
   KeyManager.$.addKeyListener( [Keyboard.CONTROL, Keyboard.C], copyText, true );
