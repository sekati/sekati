The DocumentPreloader class is an API independent shell designed to load Sekati API Document driven SWF applications In other words; DocumentPreloader should be extended and used as the Main Document Class for a preloader SWF.

The three main methods that can be overridden are:

  1. addedToStage(e:Event) - to init the visual preloader
  2. animateProgress(perc:Number) - to show load progress on a scale of 0.0 to 1.0
  3. animateComplete() - to execute a visual 'out' for the preloader. Otherwise initLoadedSwf() will be called automatically.
Note: that the API bootstrap sequence will begin when the loaded API swf is added to the stage in initLoadedSwf().
