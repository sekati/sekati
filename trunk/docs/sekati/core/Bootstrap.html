InheritanceBootstrap flash.events.EventDispatcher

Bootstrap runs thru the _sequenceChain of methods to prepare the API framework core: sekati.core.App by loading configuruations, data, stylesheets, as well as any other custom methods needed to bootstrap the application and firing the onAppConfigured event.
