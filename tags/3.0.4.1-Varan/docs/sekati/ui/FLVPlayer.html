Classpublic class FLVPlayer
InheritanceFLVPlayer CoreSprite flash.display.Sprite

FLVPlayer provides playback control interface code to be used with sekati.media.FLV.

Simply create a library asset with the necessary assets, link it to the FLVPlayer class and attach instances as needed.

See also
